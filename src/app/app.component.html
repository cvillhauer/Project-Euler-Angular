Courtney's Project Euler

My goal for this project is to solve as many of the problems from Project Euler as I can, while focusing on clean and efficient code. I also plan to spruce up the UI as I go, and to allow for at least one user-changeable variable per problem (called n in my code and notes). For example, the standard "n" for Problem 1 is 1000, but I want my solution to work no matter what the arbitrary upper limit is. Therefore, my solutions prioritize flexibility over efficiency in some cases.
